# espmeteo

С помощью этого проекта вы можете измерять давление и температуру в вашем доме. Отправка данных будет происходить в определенное время, в Telegram-бота.

## Компоненты
1. Wemos D1 mini.
2. DTH22.

## Сборка
Подключение dht22:

  Plus   -> 3v3
  
  out -> D2
  
  Minus   -> GND

## Установка 
Для установки и настройки бота:

  Скачайте  main.ino файл, запустите удобный вам редактор кода. Отредактируйте нужные переменные.

  Установите библиотеки: ESP8266WiFi.h ; DHT.h ; NTPClient.h ; WiFiUdp.h ; FastBot.h .

  Загрузите код на плату.
